# JARVIS - Python Desktop Assistant

JARVIS is a basic Python desktop assistant that accepts typed commands and gives voice responses.

## Features

- Greets the user based on time
- Tells current time and date
- Opens YouTube, Google, LinkedIn, Spotify, Notepad, Calculator, VS Code, and Command Prompt
- Searches Google
- Opens Wikipedia search pages
- Takes and saves notes
- Shows saved notes
- Tells jokes
- Shows basic system information
- Gives voice responses using pyttsx3

## Technologies Used

- Python
- pyttsx3
- datetime
- webbrowser
- os
- subprocess
- platform
- random

1. Clone or download the project.
2. Open the project folder in VS Code.
